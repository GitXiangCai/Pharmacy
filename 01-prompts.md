# Doubao

顶层是一个数组：

1. "type"分为：名解（5）、判断（2）、单选（2）、多选（2）、填空（2）、简答（10）、论述（20），括号前的为"type"的值，括号里的为"score"的值
2. "id"值为"year"-"type"-序号，其中序号为两位，如果为单位则十位添加0
3. 如果存在答案的话，则填入"answers"的值，否则"answers"、"page"值为""

请根据这个word文档整理出

示例：

[
  {
    "year": "2024",
    "type": "名解",
    "id": "2024-名解-01",
    "stem": "液体制剂",
    "answers": "",
    "score": "5",
    "page": ""
  },
  {
    "year": "2024",
    "type": "名解",
    "id": "2024-名解-02",
    "stem": "受体",
    "answers": "",
    "score": "5",
    "page": ""
  }
]

# Cursor

在"02-json文件夹"中有数个json文件，均以年份命名，一个json文件代表文件名当年的题库，我想让你根据这些json文件创造一个网站，用来刷题，所有json文件内的格式如下：

[
  {
    "year": "2024",
    "type": "名解",
    "id": "2024-名解-01",
    "stem": "液体制剂",
    "answers": "",
    "score": "5",
    "page": ""
  },
  {
    "year": "2024",
    "type": "名解",
    "id": "2024-名解-02",
    "stem": "受体",
    "answers": "",
    "score": "5",
    "page": ""
  }
]

进入网站后，上方导航栏可以选择按照年份、按照题型、随机刷题三种模式

如果是按照年份刷题，那么可以进一步选择2017~2024年，然后使用对应年份内的json数据即可

如果是按照题型刷题，那么可以选择名解、判断、单选、多选、填空、简答、论述这几种类型，分别对应json文件中的"type"字段，你需要将所有年份的都整理出来

随机刷题就是随机将随机年份的随机题目展示

主界面是数个卡片，按照苹果圆角风格，卡片上是题目（"stem"的值），左上角以灰色字体写出题目类型（"type"的值），右上角以灰色字体写出分数（"score"的值），左下角以灰色字体写出年份（"year"的值），点击卡片可翻转，背面即为答案（"answers"的值），点击下一批则替换为下一批卡片
